# vue-todo-app

Simple todo app built with Vue 3. App supports CSS preprocessor (SCSS), Linting with ESlint, and PWA.

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```

### Run your unit tests

```
npm run test:unit
```

### Lints and fixes files

```
npm run lint
```

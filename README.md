# vue-todo-app
A very simple Vue.js ( version 3 ) app which uses two way data bindings inside components ( props and custom events ), Vue transitions and internal state which is basically app data. Vuex and vue-router not used since this is a simple app and they might not be required as much.

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
